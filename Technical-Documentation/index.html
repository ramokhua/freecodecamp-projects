Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

print("Hello, World!")

Key features of Python include:

  • Easy-to-learn syntax
  • Cross-platform compatibility
  • Extensive standard library
Getting Started

To start programming in Python, you'll need to install the Python interpreter on your computer.

Here are the basic steps:

  1. Download Python from the official website
  2. Run the installer
  3. Verify the installation
python --version

This command will show you the installed Python version.

Variables and Data Types

Python has several built-in data types:

  • Numbers (int, float, complex)
  • Strings
  • Boolean
  • Lists
  • Tuples
  • Dictionaries

Variables are created when you assign a value to them:

x = 5 y = "Hello"

Python is dynamically typed, so you don't need to declare variable types.

Control Flow

Python supports the usual control flow statements found in other languages.

Conditional statements:

if x > 0: print("Positive") elif x == 0: print("Zero") else: print("Negative")

Loop statements:

for i in range(5): print(i)

While loops:

while x < 10: x += 1
Functions

Functions are defined using the def keyword:

def greet(name): print(f"Hello, {name}!")

Functions can return values using the return statement:

def square(x): return x * x

Python functions can have default arguments:

def power(base, exponent=2): return base ** exponent

Functions are first-class objects in Python, meaning they can be passed as arguments to other functions.
